- 所有项目都已在Netlify上部署

**项目1-7**

**项目8-9**

**项目10**
熟练使用状态提升进行组件通信

**项目11-13**
Navbar, Sidebar, Modal, Submenu使用React的实现
使用 `useRef` 调用DOM方法

**项目14**
使用 `useReducer` 和 `useContext` 进行全局状态管理
